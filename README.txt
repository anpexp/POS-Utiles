Este es un prototipo basado en NodeJs y Mysql para la conexión de una 
base de datos en servidor con un cliente.
El objetivo inicial es realizar la conexion con servidor y base de datos, con el tiempo se irá depurando el código a mejores prácticas y diseño.
El desarrollo permite llevar control de inventario, empleados, proveedores, clientes, y los datos resultantes de la interacción entre ellos.
